ブラウザー API を使用すると、ブラウザーの監視をカスタマイズおよび拡張できます。API の使用を開始するには、[監視するすべてのページにブラウザ エージェントが存在することを](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent)確認します。

以下は、各ブラウザ API メソッドの使用例です。

## トレースとイベント [#api-traces]

トレースやイベントを監視するには、次のような方法があります。

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        あなたがしたい場合は...
      </th>

      <th>
        これを使用
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        New Relic が自動的に追跡しないアクションを追跡する
      </td>

      <td>
        使用する `newrelic.addPageAction()`
      </td>
    </tr>

    <tr>
      <td>
        進行中の [セッショントレースにオブジェクトを追加する。](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle)
      </td>

      <td>
        使用する `newrelic.addToTrace()`
      </td>
    </tr>

    <tr>
      <td>
        独自の基準でページが終了した時点を記録する
      </td>

      <td>
        使用する `newrelic.finished()`
      </td>
    </tr>

    <tr>
      <td>
        ページ上の後続のイベントに属性名と値を追加する
      </td>

      <td>
        使用する `newrelic.setCustomAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        ページ名を設定してページビューをグループ化
      </td>

      <td>
        使用する `newrelic.setPageViewName()`
      </td>
    </tr>
  </tbody>
</table>

## エラー報告 [#api-errors]

アプリケーションのエラーを報告するには、以下の方法があります。

<table>
  <thead>
    <tr>
      <th style={{ width: "350" }}>
        あなたがしたい場合は...
      </th>

      <th>
        これを行う
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        アプリのバージョン情報でタグエラー
      </td>

      <td>
        使用する `newrelic.addRelease()`
      </td>
    </tr>

    <tr>
      <td>
        アプリの動作を停止させることなく、キャッチしたエラーや処理したエラーを記録する
      </td>

      <td>
        使用する `newrelic.noticeError()`
      </td>
    </tr>

    <tr>
      <td>
        既知のエラーを選択的に無視する
      </td>

      <td>
        使用する `newrelic.setErrorHandler()`
      </td>
    </tr>
  </tbody>
</table>

## シングル ページ アプリの追跡 (SPA のみ) [#spa-api]

シングルページアプリケーションをトラッキングするには、以下のSPA APIメソッドを使用します。

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        あなたがしたい場合は...
      </th>

      <th>
        これを行う
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        SPA[インタラクション](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)の[`actionText`](/attribute-dictionary/?event=BrowserInteraction&attribute=actionText)またはクリックされた HTML 要素のテキストを設定します
      </td>

      <td>
        使用する `newrelic.interaction().actionText()`
      </td>
    </tr>

    <tr>
      <td>
        SPAのサブコンポーネントのトレース [インタラクション](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 個別に
      </td>

      <td>
        使用する `newrelic.interaction().createTracer()`
      </td>
    </tr>

    <tr>
      <td>
        SPA [インタラクションの終了](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        使用する `newrelic.interaction().end()`
      </td>
    </tr>

    <tr>
      <td>
        現在の [インタラクションに関連する集計データ](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        使用する `newrelic.interaction().getContext()`
      </td>
    </tr>

    <tr>
      <td>
        [インタラクションを無視する](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        使用する `newrelic.interaction().ignore()`
      </td>
    </tr>

    <tr>
      <td>
        現在の [のインタラクションにバインドされた新しい API オブジェクトを返します。](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        使用する `newrelic.interaction()`
      </td>
    </tr>

    <tr>
      <td>
        [インタラクション](/attribute-dictionary/?event=BrowserInteraction) が記録される前に、名前や無視された属性などの属性を調整する。
      </td>

      <td>
        使用する `newrelic.interaction().onEnd()`
      </td>
    </tr>

    <tr>
      <td>
        ブラウザ [インタラクション](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) が終了したときに確実に保存されるようにする。
      </td>

      <td>
        使用する `newrelic.interaction().save()`
      </td>
    </tr>

    <tr>
      <td>
        現在の [インタラクションにカスタム属性を追加する](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction)
      </td>

      <td>
        使用する `newrelic.interaction().setAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        SPAルートにデフォルトの名前よりも正確な名前をつける
      </td>

      <td>
        使用する `newrelic.interaction().setCurrentRouteName`
      </td>
    </tr>
  </tbody>
</table>
