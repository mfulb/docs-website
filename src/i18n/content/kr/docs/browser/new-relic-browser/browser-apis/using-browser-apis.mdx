브라우저 API를 사용하면 브라우저 모니터링을 사용자 정의하고 확장할 수 있습니다. API 사용을 시작하려면 [모니터링하려는 모든 페이지에 브라우저 에이전트가 있는지](/docs/browser/new-relic-browser/installation/install-new-relic-browser-agent) 확인하세요.

다음은 각 브라우저 API 메소드의 사용 사례입니다.

## 추적 및 이벤트 [#api-traces]

추적 및 이벤트를 모니터링하려면 다음 방법을 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        원하는 경우...
      </th>

      <th>
        이것을 사용
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        New Relic이 자동으로 추적하지 않는 동작 추적
      </td>

      <td>
        사용 `newrelic.addPageAction()`
      </td>
    </tr>

    <tr>
      <td>
        진행 중인 [세션 추적](/docs/browser/new-relic-browser/browser-pro-features/session-traces-exploring-webpages-life-cycle) 에 개체 추가
      </td>

      <td>
        사용 `newrelic.addToTrace()`
      </td>
    </tr>

    <tr>
      <td>
        사용자 정의 기준에 따라 페이지가 완료되는 시점을 기록합니다.
      </td>

      <td>
        사용 `newrelic.finished()`
      </td>
    </tr>

    <tr>
      <td>
        페이지의 후속 이벤트에 속성 이름 및 값 추가
      </td>

      <td>
        사용 `newrelic.setCustomAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        페이지 이름을 설정하여 페이지 보기 그룹화
      </td>

      <td>
        사용 `newrelic.setPageViewName()`
      </td>
    </tr>
  </tbody>
</table>

## 오류 보고 [#api-errors]

애플리케이션의 오류를 보고하려면 다음 방법을 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350" }}>
        원하는 경우...
      </th>

      <th>
        이 작업을 수행
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        앱의 버전 정보로 오류에 태그 지정
      </td>

      <td>
        사용 `newrelic.addRelease()`
      </td>
    </tr>

    <tr>
      <td>
        앱 작동을 방해하지 않고 포착되거나 처리된 오류를 기록합니다.
      </td>

      <td>
        사용 `newrelic.noticeError()`
      </td>
    </tr>

    <tr>
      <td>
        알려진 오류를 선택적으로 무시
      </td>

      <td>
        사용 `newrelic.setErrorHandler()`
      </td>
    </tr>
  </tbody>
</table>

## 단일 페이지 앱 추적(SPA만 해당) [#spa-api]

단일 페이지 애플리케이션을 추적하려면 다음 SPA API 메소드를 사용하십시오.

<table>
  <thead>
    <tr>
      <th style={{ width: "350px" }}>
        원하는 경우...
      </th>

      <th>
        이 작업을 수행
      </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>
        SPA [상호작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 의 [`actionText`](/attribute-dictionary/?event=BrowserInteraction&attribute=actionText) 또는 클릭한 HTML 요소의 텍스트를 설정합니다.
      </td>

      <td>
        사용 `newrelic.interaction().actionText()`
      </td>
    </tr>

    <tr>
      <td>
        SPA [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 의 하위 구성 요소를 별도로 추적
      </td>

      <td>
        사용 `newrelic.interaction().createTracer()`
      </td>
    </tr>

    <tr>
      <td>
        SPA [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 종료
      </td>

      <td>
        사용 `newrelic.interaction().end()`
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 과 관련된 데이터 집계
      </td>

      <td>
        사용 `newrelic.interaction().getContext()`
      </td>
    </tr>

    <tr>
      <td>
        [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 무시
      </td>

      <td>
        사용 `newrelic.interaction().ignore()`
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 에 바인딩된 새 API 개체 반환
      </td>

      <td>
        사용 `newrelic.interaction()`
      </td>
    </tr>

    <tr>
      <td>
        [상호 작용](/attribute-dictionary/?event=BrowserInteraction) 이 기록되기 전에 이름 및 무시된 속성과 같은 속성 조정
      </td>

      <td>
        사용 `newrelic.interaction().onEnd()`
      </td>
    </tr>

    <tr>
      <td>
        브라우저 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 이 끝날 때 저장되는지 확인
      </td>

      <td>
        사용 `newrelic.interaction().save()`
      </td>
    </tr>

    <tr>
      <td>
        현재 [상호 작용](/docs/using-new-relic/welcome-new-relic/get-started/glossary/#interaction) 에 사용자 정의 속성 추가
      </td>

      <td>
        사용 `newrelic.interaction().setAttribute()`
      </td>
    </tr>

    <tr>
      <td>
        SPA 경로에 기본 이름보다 더 정확한 이름 지정
      </td>

      <td>
        사용 `newrelic.interaction().setCurrentRouteName`
      </td>
    </tr>
  </tbody>
</table>
